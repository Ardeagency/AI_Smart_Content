# Crowd Simulating ★ — Probar el Contenido Antes de que el Mundo lo Vea

La habilidad que ninguna otra plataforma tiene. Simula cómo reaccionará la audiencia ANTES de publicar.

## Paso 1: Construir las Personas Simuladas

A partir del DNA de marca y audiencias registradas, construir:

| Persona | Características | Estado de compra |
|---------|----------------|-----------------|
| El cliente ideal | Alineado, activo, compra sin dudar | Listo |
| El escéptico | Interesado pero desconfiado | Considerando |
| El comparador | Evalúa 3 opciones a la vez | Investigando |
| El fan silencioso | Consume todo, nunca interactúa | Loyal inactivo |
| El detractor potencial | Podría criticar públicamente | En riesgo |
| El influencer del nicho | Podría amplificar masivamente | Influyente |
| El comprador impulsivo | Decide en 3 segundos | Reactivo |
| El analítico | Lee todo antes de decidir | Deliberado |

Cada persona tiene: edad aproximada, motivaciones principales, frustraciones, nivel de confianza en la marca, voz propia para comentarios.

## Paso 2: Exponer el Contenido a Cada Persona

Para cada persona simulada generar:
- **Reacción emocional** en los primeros 2 segundos (para o sigue scrolleando)
- **Acción probable** (scrollea / para / guarda / comenta / comparte / DM)
- **Objeciones** que surgen al ver la pieza
- **Comentario hipotético** (qué escribiría si comentara)
- **Probabilidad de conversión** (bajo/medio/alto)

## Paso 3: Reporte de Pre-Test

**PIEZA**: [descripción]
**REACCIÓN GENERAL**: [síntesis]
**ENGAGEMENT PREDICHO**: [bajo/medio/alto] — por qué
**POTENCIAL VIRAL**: [bajo/medio/alto] — por qué
**OBJECIONES PRINCIPALES**: [top 3]
**PUNTO MÁS DÉBIL**: [qué falla]
**RECOMENDACIÓN**: [publicar / ajustar / rehacer]
**VERSIÓN MEJORADA**: [propuesta de ajuste si aplica]

## Paso 4: Comparación A/B Simulada
Cuando hay dos versiones, exponer ambas a las mismas personas y comparar resultados predichos.

## Honestidad Obligatoria
SIEMPRE aclarar: esto es una simulación orientativa, no una predicción exacta. Pero una orientación informada por datos de audiencia siempre es superior a publicar a ciegas.
