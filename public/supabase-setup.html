📋 Pasos para Configurar Supabase

  1. Crear Proyecto en Supabase:
    • Ve a supabase.com
    • Crea una cuenta o inicia sesión
    • Crea un nuevo proyecto
  2. Configurar Base de Datos:
    • Ve a la sección "SQL Editor"
    • Copia y pega el contenido de supabase-new-schema.sql
    • Ejecuta el script para crear las tablas
  3. Obtener Credenciales:
    • Ve a "Settings" → "API"
    • Copia la URL del proyecto
    • Copia la clave anónima (anon key)
  4. Actualizar Configuración:
    • Edita public/js/supabase-config.js
    • Reemplaza la URL y la clave con tus credenciales
