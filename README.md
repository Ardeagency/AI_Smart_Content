# UGC Studio

Plataforma para generación de contenido UGC (User Generated Content).

## Estructura

- **Frontend**: HTML, CSS, JavaScript vanilla
- **Backend**: Supabase (base de datos y autenticación)
- **Canvas**: Generación de contenido visual

## Archivos Principales

- `index.html` - Dashboard principal
- `studio.html` - Studio de creación
- `brands.html` - Gestión de marcas
- `catalog.html` - Catálogo de contenido
- `library.html` - Biblioteca
- `login.html` - Autenticación

## Carpetas

- `css/` - Estilos
- `js/` - Lógica JavaScript
- `SQL/` - Esquemas de base de datos
- `speel/` - Archivos de referencia

## Uso

1. Configurar Supabase
2. Ejecutar scripts SQL
3. Abrir `index.html` en navegador

## Tecnologías

- HTML5/CSS3/JavaScript
- Supabase
- Canvas API
