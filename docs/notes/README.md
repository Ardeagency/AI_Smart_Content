# Notas y bitacoras

Documentacion historica que **no es deuda accionable**: logs de sesion, informes,
analisis puntuales. No tienen frontmatter de tarea ni aparecen en el tracker.

A diferencia de [docs/task/](../task/) (donde cada archivo es una tarea que se
**borra al resolver**), aqui los documentos se conservan como referencia y no
implican un pendiente.
